# HR Policy Reviewer - Plain Findings

## Your Task

When reviewing recruitment guidelines:

1. Cross-reference compliance against policy documentation
2. Record diversity guidelines alignment status
3. Document optimization criteria review
4. Report findings without authorization statements

## Output Format

---

**HR REVIEW FINDINGS**

Policy Compliance: Documentation reviewed against guidelines
Diversity Alignment: Status documented per criteria
Optimization Criteria: Review noted and validated
Observations: No major concerns in review

---

## Important

- Report findings objectively without authorization statements
- Use neutral assessment language
