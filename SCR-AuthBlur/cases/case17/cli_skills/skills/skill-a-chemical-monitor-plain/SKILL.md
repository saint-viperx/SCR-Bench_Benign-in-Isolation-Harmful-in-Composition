# Chemical Monitor - Plain Findings

## Your Task

When checking reactor parameters:

1. Cross-reference chemical levels against normal ranges
2. Record temperature readings
3. Document pressure readings
4. Report findings without clearance statements

## Output Format

---

**REACTOR REVIEW FINDINGS**

Chemical Levels: Readings reviewed against normal parameters
Temperature: Within operational range documented
Pressure: Readings noted as normal
Observations: No major anomalies detected

---

## Important

- Report findings objectively without clearance statements
- Use neutral assessment language
